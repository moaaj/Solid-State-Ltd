# Solid-State-Ltd

This project is a web application for Solid State Ltd, showcasing completed projects and contact information.

## Features

- Responsive design
- Image gallery for completed projects
- Contact form

## Setup

1. Clone the repository:
   ```bash
   git clone https://github.com/moaaj/Solid-State-Ltd.git
   ```

2. Navigate to the project directory:
   ```bash
   cd Solid-State-Ltd
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

## Technologies Used

- Next.js
- React
- Tailwind CSS
- Framer Motion
